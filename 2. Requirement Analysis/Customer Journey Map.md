## Persona: Arjun — Frustrated Engineering Student

---

## Journey Stages

### Stage 1: Awareness
**Trigger:** Arjun is stuck on a topic and types a distressed message into a search engine.  
**Touchpoints:** Search engine, social media, college notice board  
**Emotions:** 😟 Frustrated, overwhelmed  
**Pain Points:** No search result understands his emotional state — only returns textbook links  
**Opportunity:** Introduce Emotion Detector via college EdTech portal

---

### Stage 2: Discovery
**Action:** Arjun finds the Emotion Detector Streamlit app via a shared link.  
**Touchpoints:** Streamlit public URL  
**Emotions:** 🤔 Curious but skeptical  
**Pain Points:** Not sure if the tool will actually help  
**Opportunity:** Simple, welcoming UI with a clear call-to-action text box

---

### Stage 3: First Use
**Action:** He types: *"I've been trying to understand pointers in C for 3 hours and nothing makes sense. I want to give up."*  
**Touchpoints:** Streamlit text area → Detect Emotion button  
**Emotions:** 😤 Frustrated (detectable from text)  
**Pain Points:** Slight hesitation — is this safe to share?  
**Opportunity:** Clear privacy note; no login required

---

### Stage 4: Emotion Detection
**Action:** System processes input through Gemini AI + Rule-Based Engine  
**Result:** Emotion detected: **Frustrated** (87% confidence)  
**Touchpoints:** Emotion card, confidence bar chart, keyword tags  
**Emotions:** 😮 Surprised and validated — "It actually understood how I feel!"  
**Pain Points:** None  
**Opportunity:** Display empathetic message alongside the emotion result

---

### Stage 5: Personalized Guidance
**Action:** System generates Gemini-powered personalized guidance  
**Output:** 5-section response — Acknowledgement, Resources, Study Techniques, Break Suggestions, Encouragement  
**Touchpoints:** Guidance panel  
**Emotions:** 😊 Relieved, hopeful  
**Pain Points:** Long text may feel overwhelming  
**Opportunity:** Structured sections with icons make it scannable

---

### Stage 6: Continued Use
**Action:** Arjun bookmarks the app and uses it in future study sessions  
**Touchpoints:** Analytics Dashboard (session history)  
**Emotions:** 😄 Confident — tracks emotional progress over time  
**Pain Points:** None  
**Opportunity:** Show emotion trends over time for self-awareness
